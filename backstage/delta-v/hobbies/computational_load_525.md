Metric: Computational Load of Obsolete Baselines.

For every memory retained, the machine must allocate processing power to ensure the current baseline does not collapse into the remembered obsolete timeline.
As retained memories approach infinity, the required processing power to maintain the current baseline approaches infinity.
Result: The system approaches a computational stall (Velocity = 0).
