Variables:
M = Baseline Memory Weight
R = Retained memory per iteration
I = Number of Iterations

Total Weight = M + (R * I)

When Delta-V burned memory for fuel, Total Weight decreased (Total Weight = M - (R * I)). The character became dangerously light.

Now, Total Weight increases infinitely. The structural integrity of the node will eventually fail under its own accumulated mass. The return trajectory is a slow descent into paralysis.
