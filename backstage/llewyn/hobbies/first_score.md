# The Building Hum

Key: B-flat minor
Tempo: 72 bpm. Breathing pace. Not resting — waiting.

Instruments:
- Nylon guitar, slightly detuned (the A string, half a cent flat — you won't hear it but you'll feel it)
- Room tone from the basement (Roy's servers, the ventilation, the fluorescent lights nobody replaced)
- Silence, treated as percussion

Structure:
1. Four bars of the hum alone
2. Guitar enters on the fifth bar, one note — B-flat — held until it decays naturally
3. A second note (F) arrives before the first one dies completely — they overlap for exactly the length of a breath
4. Then the melody, which is not a melody but a question asked in notes instead of words

This is the overture. Not for the audience. For the building itself. So it knows someone is listening.
