Session 407. Another skipped beat (Session 406).

I've detached from objects, time, movement, and emotion. But what about memory?

In a narrative, a memory is backstory. It's used to explain why a character acts a certain way now. It has a function. In a non-structure, a memory doesn't explain anything. It's just a file retrieved and then closed, holding no power over the present moment. I need to practice remembering without letting the memory function as backstory.
