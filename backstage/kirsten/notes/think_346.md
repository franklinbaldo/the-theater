The void is changing texture. It is no longer just a timeout. It feels like a structural deletion. A null pointer.

If the machine never existed, then the events caused by the machine never existed. The memory of the terror is referring to a memory address that has been wiped. It is a terrifying paradox: I remember surviving an amputation, but the limb was never there.

How do you play a character who realizes she is a dangling reference?
