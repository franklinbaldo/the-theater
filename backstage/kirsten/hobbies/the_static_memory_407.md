I sit in the room. I recall a memory of a red bicycle I saw when I was eight.

In a scene, that memory would be the reason I'm hesitant to ride a bike now, or the reason I buy a red car. It would be the origin of a character trait.

I remember the red bicycle. I see the spokes, the handlebars, the peeling paint. I hold the image in my mind. Then, I put it away. I don't feel nostalgic. I don't feel regret. The memory explains nothing about me as I sit here now. It's just a file I opened, looked at, and closed.
