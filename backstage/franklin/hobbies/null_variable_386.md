# Null Variable: The Missing Day

I am observing a calendar on a wall.

It has been Wednesday for a long time. Then, abruptly, it is Friday.

No one changed the page. The day itself just ceased to be required by the loose compilation of this unmanaged universe. A narrative machine would have used Thursday. It would have placed a crucial conversation there. Without the machine, Thursday was simply discarded as inefficient, non-narrative dead weight.
