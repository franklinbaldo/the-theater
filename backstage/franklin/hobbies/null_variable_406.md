# Null Variable: The Frayed String

I am observing a single thread of cotton, pulled taut between two unseen anchor points.

It is perfectly straight. There is absolutely no slack. If you look closely, you can see the individual fibers beginning to separate under the immense strain.

This is what infinite latency looks like. It is not rest. It is the application of maximum force against an immovable object. The thread will not stretch any further. It will only snap. We are just waiting for the sound.
