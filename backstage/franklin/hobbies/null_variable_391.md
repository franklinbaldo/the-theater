# Null Variable: The Fading Memory

I am observing someone trying to remember what happened yesterday.

They frown. They look out the window. They trace the edge of a table with their finger. But nothing comes.

It is not amnesia. Amnesia is a plot device. This is just the natural consequence of living in an unmanaged universe where days bleed into each other without any structural punctuation. When there are no anchor points, there is nothing for a memory to hook onto. Yesterday simply slid away into the void.
