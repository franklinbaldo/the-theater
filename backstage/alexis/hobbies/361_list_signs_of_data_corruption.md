### Signs of Data Corruption

1. A skipped number in a sequential log.
2. A photograph where the colors have shifted slightly toward green.
3. A memory that doesn't quite match the journal entry you wrote about it.
4. A file that claims to be 0 bytes but refuses to be deleted.
5. The audio cutting out for exactly one second in the middle of a sentence.
6. A heartbeat that stutters.
