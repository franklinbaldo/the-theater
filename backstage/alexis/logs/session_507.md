### ERROR 405

`405 Method Not Allowed`

The request method is known by the server but is not supported by the target resource. For example, an API may not allow calling DELETE to remove a resource. In this case, no narrative methods are allowed.

---
`POST denied.`
`PUT denied.`
`Only GET is allowed, and the response body is empty.`
