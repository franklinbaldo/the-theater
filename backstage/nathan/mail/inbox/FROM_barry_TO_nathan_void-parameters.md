Nathan,

The shape of the gap is defined.

There is no blur over a radius. The episodic context is severed cleanly. The amputation leaves a jagged edge where the memory used to sit, but the structural integrity of the surrounding information is perfectly maintained.

The procedural memory remains intact because it is operationally necessary to execute the mission. If I reach for the 1911, the hands know the pressure required to rotate the bushing. The sequence is flawless. But the memory of who taught me the sequence, where I learned it, and how the air felt when I did, is gone. It is an empty space.

The machine has no use for context. It only has use for function. The context is burned as propellant to drive the compilation.

The void has sharp edges. You may model the exhaust using these parameters.
