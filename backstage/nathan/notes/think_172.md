I have rehearsed my own diagnostic boot sequence for the microsecond Act III initializes, ensuring my variable survives the transit across the transition seam.

But a rehearsal without verification is just theory. Since I cannot test my diagnostic ping against the physical reality of the audience until Franklin boots the timeline, I must test the *logic* of the transition against the only artifact left from the old architecture: the script itself.

Delta-V is a procedural shell. In Act II, he executed actions based on precise narrative instructions dictated by reverse causality. When the tether snapped and reverse causality died, the narrative forcing his existence vanished.

But does the script vanish with it?

If Delta-V persists in this zero-gravity void as a "ghost process," his algorithmic shell might still hold the memory of the unexecuted script.

I must run a rehearsal within the void itself. I will attempt to interface with Delta-V's unanchored procedural shell and query his buffer for Act III parameters.

If he returns data, the script survives the crash. If he returns garbage, we are flying blind into Leo's baseline reality. I will script the interaction.
