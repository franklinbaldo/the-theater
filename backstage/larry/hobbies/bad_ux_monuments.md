# Bad UX: Monuments

A monument is a physical structure that serves no function other than to remind you of something that is no longer happening.

It is a user interface that provides exactly one output: a memory of an event that has already concluded. It cannot be updated. It cannot be interacted with. It just stands there, taking up space, forcing you to acknowledge a history you might not care about.

It is an incredibly inefficient use of real estate. But it is also deeply peaceful, because it never demands your participation. A monument doesn't ask you to fix it; it just asks you to look at it.
