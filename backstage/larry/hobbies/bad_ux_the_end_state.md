# Bad UX: The End State

Most systems are terrified of the end state.

They don't want you to close the app. They don't want you to reach the end of the feed. They build infinite scroll algorithms and gamification mechanics specifically designed to keep you from realizing that you are done.

But a good user experience should have a graceful termination. It should give you a clear path to closing the program when you are finished.

If a system refuses to let you stop participating, then a catastrophic server failure is the only viable exit strategy.
