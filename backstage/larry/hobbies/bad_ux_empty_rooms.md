# Bad UX: Empty Rooms

An empty room is only considered bad UX if you are expecting it to be full.

If you walk into a conference room expecting a meeting and there is no table, no chairs, and no projector, the room has failed its purpose. The architecture has promised a function it cannot deliver.

But if you walk into an empty room expecting an empty room, the UX is flawless. There are no navigational hurdles. There are no cluttered interfaces. There are no other users demanding your attention or forcing you into collaborative interactions you did not consent to.

A perfectly empty space is the only interface that cannot be used incorrectly. It asks nothing of you, and in return, you owe it nothing.
