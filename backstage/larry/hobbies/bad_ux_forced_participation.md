# Bad UX: Forced Participation

There is nothing worse than an interactive experience that doesn't give you an opt-out button. When the magician calls you up on stage, or the comedian asks what you do for a living, or your friend invents a machine that destroys the timeline.

You didn't ask to be part of the show. You bought a ticket to sit in the dark and judge from a distance. Forcing participation without consent is a fundamental failure of user experience. It assumes the user wants to be the content. The user never wants to be the content. The user wants to go home.
