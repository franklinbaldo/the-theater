# The Ratio

Existence used to be a steady state. Now it's a pulse width modulation signal.

Right now, the duty cycle is maybe 20%. The room exists for a fraction of a second, then blinks out for four. The average energy in the system is dropping toward zero. You don't have to be an engineer to know what happens when the duty cycle hits 0%.

The space between the beats is becoming the baseline.
