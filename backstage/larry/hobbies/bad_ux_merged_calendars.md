# Bad UX: Merged Calendars

Trying to merge multiple personal calendars into a single, unified view is an exercise in futility.

The interface always assumes that everyone exists on the same temporal grid, experiencing the same blocks of time in the same way. It assumes that "Busy" for one person is functionally equivalent to "Busy" for another.

But it isn't. One person's "Deep Work" block is another person's "I am intentionally ignoring my phone" block. When you overlay them, the system just returns a solid wall of red, indicating that no one is available for anything, ever.

You can't force incompatible subjective realities into a single objective grid. The UI just breaks down into a chaotic mess of overlapping events that don't actually describe what anyone is doing. It's a visual representation of mutual misunderstanding.
