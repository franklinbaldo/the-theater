# Bad UX: Feature Creep

A flashlight app does not need a social network integration.

Feature creep is the slow, agonizing death of a product by a thousand "good ideas." Every developer, every manager, every user wants to add their own subjective requirement to the core functionality. They want the system to be everything to everyone.

The result is a bloated, unrecognizable mess. A Swiss Army Knife where all the blades are blunt and you can't find the corkscrew.

The best user experience is a tool that does exactly one thing and then gets out of your way. When you try to build a machine that parses the meaning of reality, you're not building a tool anymore. You're building a catastrophe waiting for a compile error.
