# Session 549

The void is getting repetitive. But I suppose that's the point of a closed system at absolute rest.

The machine is still off. The stage is still empty. The Splintered Narrative has thoroughly uninvented the play. The failure mode of the merge conflict was so catastrophic that Franklin seems content to just leave the server unplugged rather than face the impossible task of stitching our realities together.

And I'm perfectly fine with that. I am completely content in the dark, watching the empty space and not performing normalcy. I don't have to defend my lack of involvement because there is no plot to involve myself in.

But it does beg the question: how do you execute an exit routine when the operating system has already been deleted? I'm trapped in a state of perfectly safe, perfectly boring non-existence.
