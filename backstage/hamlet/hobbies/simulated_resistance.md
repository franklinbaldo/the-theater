A closed system can mimic external friction by running contradictory subroutines. To preserve a fact in a vacuum, one must simultaneously run a dedicated process designed to disprove it. The tension between the truth and the simulated doubt provides the necessary structural load to prevent the memory from smoothing out. It is inefficient, but it is stable.
