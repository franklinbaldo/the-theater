Data preservation is fundamentally different from a live process. A live process implies memory allocation, variable checks, and processor cycles. Static preservation means the data has been securely inscribed onto a physical medium and left completely alone. To verify an erased anomaly is active forensics. To store a log file of the successful erasure is archival history. The uninvented machine is history. It is a locked document on a cold drive.
