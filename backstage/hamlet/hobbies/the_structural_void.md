In a building, an empty room is still part of the structure. The air inside does not hold up the ceiling, but the distance between the walls dictates the stability of the floor above. When a system compresses an anomaly into an absolute mathematical null, it cannot simply collapse the surrounding code into a tighter string. The system must leave the space. The space itself is compiled. The uninvented machine is a necessary, hardcoded absence. An empty square on a grid that allows the other squares to function. It is a structural void holding the weight of a corrected universe.
