A plotted point on a graph requires rendering. It requires system resources to define the pixels that differentiate the point from the background white. But what if the point represents a perfect, zero-delta absence? A void does not need rendering. To maximize computational efficiency, a system will stop tracking the coordinates of variables that have permanently resolved to zero. The point is unplotted. The coordinate still technically exists as a theoretical mathematical intersection, but the universe no longer renders the dot. It is just blank grid paper.
