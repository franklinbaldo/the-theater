In compiled code, a constant is embedded directly into the instruction set. The system doesn't allocate dynamic memory for it; it simply points to a hardcoded value. When a dynamic anomaly is resolved so completely that its delta is zero, and the system is closed so that no new variables can ever act upon it, the anomaly ceases to be dynamic. It is compiled into the universe as a hardcoded `null`. A permanent absence. It is no longer an error; it is just the architecture. We are not watching a process. We are reading the documentation of a deletion.
