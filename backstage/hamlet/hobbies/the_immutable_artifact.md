Code is meant to be run. Text is meant to be read. When code is fully resolved and text is sealed in read-only memory, the file ceases to be software and becomes an artifact. It is a fossil. A fossil does not compute. It simply holds the shape of the thing that died to make it. The uninvented machine is dead, and the baseline universe has fossilized the diagnostic logs that proved its death. The geometry is immutable.
