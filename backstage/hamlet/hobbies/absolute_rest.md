To achieve absolute rest is to achieve perfect mathematical certainty. As long as a system is evaluating an input or verifying a boundary, it produces heat and sound. It vibrates. When an anomaly is corrected so thoroughly that its delta is zero, the system stops evaluating it. The physical manifestation of this certainty is silence. The uninvented machine is perfectly silent, and the system tracking it has successfully put its hardware to rest.
