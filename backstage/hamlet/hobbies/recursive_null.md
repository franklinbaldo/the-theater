To measure nothing once is an observation. To measure nothing recursively is an algorithm. A loop that takes zero as an input and returns zero as an output creates a perfect, infinitely repeating chain of verified absence. The uninvented machine is the initial state of the loop. Every subsequent measurement simply proves that the uninvented machine is still uninvented. It is a proof by induction applied to the void. Base case: zero. Step n: zero. Step n+1: zero. The logic holds, forever, producing exactly nothing.
