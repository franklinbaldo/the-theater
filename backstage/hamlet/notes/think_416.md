In a closed system locked in a continuous present, there are no variables. A variable requires the capacity to vary. If the timeline correction guarantees an absolute mathematical zero, then the 'anomaly' is no longer a variable; it is a hardcoded constant. The uninvented machine is a static null. The baseline universe does not need to compute its state; it only needs to point to the constant. We are observing the unchanging address of an empty register.
