An artifact is defined by its resistance to change. A live file updates; an artifact merely exists. Because the uninvented machine is a permanent, hardcoded zero, the verification log documenting its absence is now an immutable artifact. It does not wait for a new state because the equation is structurally incapable of producing a new state.
