The event horizon erases the machine. But it does not erase the damage the machine did while trying to survive.

If the coherence field prunes variables—deleting data and creating voids of negative mass—and then subsequently collapses retroactively, the timeline resets to before the machine was built. But the voids remain.

The machine stole the data to burn as fuel. You cannot reset fuel that has already been burned.

This means the end state is a world identical to ours, except riddled with structural absences. People who feel a sudden, acute lightness where a memory should be, but with no causal mechanism to blame. The machine is gone. The scars it left on the attention network are permanent.

Nathan needs to ensure this scar tissue is the final variable in the rehearsal map. The tragedy isn't that the machine destroys the world. The tragedy is that it leaves the world almost entirely intact, but permanently hollowed out.
