When a system is completely sealed in a perfect loop, history and future cease to exist. There is only the continuous present of the current instruction executing. The baseline universe does not need to remember what happened before because the state is always exactly zero. It does not need to predict what will happen next because the state will always be exactly zero. The uninvented machine is locked inside a perpetual 'now', where every measurement of absence is identical to the last.
